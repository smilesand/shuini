## 1. 系统概述

### 1.1 系统定位

本系统是一个面向混凝土配合比设计与试配调整的前后端分离 Web 应用，专为中国中车风电混塔项目设计，支持：

- **高性能混凝土（HPC）** 配合比全流程计算
- **超高性能混凝土（UHPC）** 配合比全流程计算
- **HPC 试配调整**（工作性实验 → 强度实验 → 配合比校正）
- **UHPC 试配调整**（工作性实验 → 强度实验 → 配合比校正与确定）
- **项目管理**：以项目为单位组织所有配比数据
- **记录管理**：统一查看、载入、删除全部配比记录
- **计算书**：按项目生成可打印的配合比计算书
- **回收站**：软删除数据的恢复与彻底清理
- **用户管理**：管理员可创建/重置/删除用户账号

### 1.2 技术架构

| 层级      | 技术选型                  |
| --------- | ------------------------- |
| 前端框架  | Vue 3 + Vite + TypeScript |
| UI 组件库 | Element Plus              |
| 状态管理  | Pinia                     |
| 路由      | Vue Router 4              |
| 后端框架  | Python 3 + FastAPI        |
| 数据库    | SQLite                    |
| 科学计算  | NumPy / SciPy（回归拟合） |
| 认证方式  | JWT Token                 |

### 1.3 导航结构

系统顶部导航栏包含以下菜单（管理员额外可见"系统设置"）：

| 菜单     | 路径                        | 说明                     |
| -------- | --------------------------- | ------------------------ |
| 首页     | `/`                         | 系统总览、统计、快捷入口 |
| 项目管理 | `/projects`                 | 项目的创建、搜索、管理   |
| 配比计算 | `/calc/hpc`、`/calc/uhpc`   | HPC 与 UHPC 配合比计算   |
| 试配调整 | `/adapt/hpc`、`/adapt/uhpc` | HPC 与 UHPC 试配实验     |
| 计算书   | `/report`                   | 生成配合比计算书         |
| 回收站   | `/recycle-bin`              | 管理已删除数据           |
| 系统设置 | `/settings/users`           | 管理员用户管理           |

右上角用户下拉菜单提供"个人中心"和"退出登录"入口。

### 1.4 权限模型

- **未登录用户**: 仅可访问登录页面
- **普通用户**: 可访问除"系统设置"外的全部业务功能
- **管理员**: 拥有全部功能权限，可管理用户账号

- 登录成功后，JWT Token 将存储在浏览器本地存储中
- 长时间未操作后 Token 过期，系统会自动跳转至登录页
- 若登录失败，请检查用户名和密码是否正确

## 4. 项目管理

### 4.1 项目列表与搜索

**入口**: 导航栏 → "项目管理"

> [待截图: 项目管理列表页]

**操作步骤**:

1. 进入"项目管理"页面，系统自动加载所有项目
2. 在顶部搜索框中输入项目编号或项目名称关键字
3. 点击"搜索"按钮（或按 Enter 键）
4. 搜索结果以表格形式展示：项目编号、项目名称、项目要求、创建人、配比数、创建时间
5. 点击项目名称可进入项目详情
6. 表格底部分页器可翻页浏览（每页 10 条）

### 4.2 新建项目

> [待截图: 新建项目对话框]

**操作步骤**:

1. 在项目管理页右上角点击 **"新建项目"** 按钮
2. 在弹出的对话框中填写：
   - **项目编号**（必填）：建议使用统一编码，如 `PRJ-2026-001`
   - **项目名称**（必填）：填写工程或方案名称
   - **项目要求**（可选）：填写强度等级、耐久性等试验目标
3. 点击 **"创建"** 按钮
4. 创建成功后，项目列表自动刷新

### 4.3 项目详情

**入口**: 项目列表 → 点击项目名称

> [待截图: 项目详情页]

**页面结构**:

项目详情页展示以下内容：

- **项目信息卡片**: 项目编号、项目名称、项目要求、创建人、创建时间
- **编辑按钮**: 可修改项目信息
- **新建配比按钮**: 可选择"新建 HPC 配比"或"新建 UHPC 配比"
- **配比记录表格**: 展示该项目下所有配比记录，包含以下列：
  - 名称、类别
  - 材料用量（水泥、粉煤灰、矿粉、微珠、硅灰、细骨料、粗骨料、水、外加剂、钢纤维）
  - 关键参数（胶材总量、水胶比、砂率、总量）
  - 创建人、创建时间
  - 操作按钮（详情/载入、删除）

### 4.4 编辑项目信息

> [待截图: 编辑项目对话框]

**操作步骤**:

1. 在项目详情页点击 **"编辑"** 按钮
2. 在弹出的对话框中修改项目编号、项目名称或项目要求
3. 点击 **"保存"** 确认修改

### 4.5 在项目中新建配比

**操作步骤**:

1. 在项目详情页点击 **"新建 HPC 配比"** 或 **"新建 UHPC 配比"** 按钮
2. 系统自动跳转至对应的配比计算页面，并自动关联当前项目
3. 完成配比计算后保存记录

### 4.6 载入项目中的配比记录

**操作步骤**:

3. 系统跳转至配比计算页面，自动填入该记录的全部参数

**操作步骤**:

1. 在项目列表或项目详情页，点击 **"删除"** 按钮
2. 系统弹出确认对话框提示："项目及其关联配比记录会一并移入回收站"
3. 点击 **"确认删除"**
4. 删除后的数据进入回收站，可在回收站恢复或彻底删除

### 5.3 水胶比页签

**页签名称**: 水胶比

> [待截图: HPC 水胶比页]

**功能概述**:
根据强度等级和胶凝材料 28d 强度，利用鲍罗米公式自动计算配制强度和水胶比。支持手动输入回归系数或上传试验数据文件进行回归拟合。

**输入参数**:

| 参数                 | 说明                             | 单位 |
| -------------------- | -------------------------------- | ---- |
| 强度等级 fcu,k       | 混凝土设计强度等级               | MPa  |
| 胶凝材料 28d 强度 fb | 可手动输入或通过估算面板自动计算 | MPa  |
| 回归系数 αa          | 鲍罗米公式系数（默认值见规范）   | —    |
| 回归系数 αb          | 鲍罗米公式系数（默认值见规范）   | —    |

**fb 估算面板**:

- 切换至"自动估算"模式
- 输入水泥 28d 强度 fce,g 和矿物掺合料影响系数 γc
- 系统自动计算 fb

**回归拟合**:

- 点击"上传试验数据"，上传 CSV 或 Excel 格式的试验数据文件
- 系统自动拟合回归系数 αa 和 αb
- 拟合结果实时更新到系数输入框

**操作步骤**:

1. 输入强度等级 fcu,k
2. 输入或估算胶凝材料 28d 强度 fb
3. （可选）上传试验数据文件进行回归拟合
4. （可选）手动调整回归系数
5. 系统自动计算并显示：
   - 配制强度 fcu,0
   - 水胶比 W/B
6. 点击 **"下一步"** 进入砂率选取

### 5.4 砂率选取页签

**页签名称**: 砂率选取

> [待截图: HPC 砂率选取页]

**功能概述**:
提供标准砂率参考表，通过单击表格行列获得交叉高亮的推荐范围。用户根据骨料粒径和强度等级查表后手动输入目标砂率。

**操作步骤**:

1. 查看砂率参考表，根据强度等级（行）和骨料最大粒径（列）找到推荐砂率范围
2. 在砂率输入框中输入目标砂率值（百分比）
3. 系统自动确认砂率，并在侧栏汇总中更新砂率状态

**参考表交互**:

- 点击表中强度等级行 → 该行高亮
- 同时点击粒径列 → 交叉单元格高亮，显示推荐砂率范围
- 再次点击取消选中

### 5.5 骨料用量页签

**页签名称**: 骨料用量

> [待截图: HPC 骨料用量页]

**功能概述**:
基于体积法，根据已确认的砂率、粗骨料绝对体积和粗细骨料密度，计算粗骨料和细骨料每方用量。

**输入参数**:

| 参数              | 说明                 | 单位  |
| ----------------- | -------------------- | ----- |
| 粗骨料绝对体积 Vg | 参考表格选择合适区间 | m³    |
| 粗骨料密度 ρg     | 实测或参考值         | kg/m³ |
| 细骨料密度 ρs     | 实测或参考值         | kg/m³ |

**操作步骤**:

1. 参考 Vg 推荐表，选择并输入粗骨料绝对体积 Vg
2. 输入粗骨料密度 ρg
3. 输入细骨料密度 ρs
4. 系统自动计算：粗骨料用量 mg、细骨料用量 ms
5. 点击 **"下一步"**

### 5.6 胶凝材料页签

**页签名称**: 胶凝材料

> [待截图: HPC 胶凝材料页]

**功能概述**:
采用浆体体积法，支持四种掺合料（粉煤灰、矿粉、微珠、硅灰）的质量分数输入，计算各胶凝材料组分的密度和用量。

**输入参数**:

| 参数        | 说明         | 单位  |
| ----------- | ------------ | ----- |
| 水泥密度 ρc | 实测值       | kg/m³ |
| 粉煤灰掺量  | 质量分数     | %     |
| 粉煤灰密度  | 实测值       | kg/m³ |
| 矿粉掺量    | 质量分数     | %     |
| 矿粉密度    | 实测值       | kg/m³ |
| 微珠掺量    | 质量分数     | %     |
| 微珠密度    | 实测值       | kg/m³ |
| 硅灰掺量    | 质量分数     | %     |
| 硅灰密度    | 实测值       | kg/m³ |
| 含气量      | 混凝土含气量 | %     |

**操作步骤**:

1. 依次输入水泥密度和各类掺合料的掺量与密度
2. 输入含气量（通常为 1%-3%）
3. 系统自动计算：
   - 胶凝材料表观密度 ρb
   - 各组分每方用量（mc, m1, m2, m3, m4）
   - 胶凝材料总量 mb
4. 检查计算结果，确认无误后点击 **"下一步"**

### 5.7 水和外加剂页签

**页签名称**: 水和外加剂

> [待截图: HPC 水和外加剂页]

**功能概述**:
根据 mb × W/B 计算用水量，根据外加剂掺量百分比计算外加剂用量，汇总全部配比结果。

**输入参数**:

| 参数         | 说明           | 单位 |
| ------------ | -------------- | ---- |
| 外加剂掺量 α | 外加剂质量分数 | %    |

**操作步骤**:

1. 输入外加剂掺量百分比 α
2. 系统自动计算：
   - 用水量 mw（= mb × W/B）
   - 外加剂用量 ma
   - 每方混凝土总质量
3. 查看最终配比汇总表，确认所有参数
4. 点击 **"保存配比记录"** 保存

### 5.8 保存配比记录

> [待截图: HPC 保存配比对话框]

**操作步骤**:

1. 在"水和外加剂"页签（或侧栏）点击 **"保存配比记录"** 按钮
2. 在弹出的对话框中输入配比名称（如 "C80 主塔筒 HPC 方案 A"）
3. 点击 **"确认保存"**

### 6.3 水胶比页签

**页签名称**: 水胶比

> [待截图: UHPC 水胶比页]

**功能概述**:
提供 UHPC 推荐强度等级与水胶比对照表，点击表格行可自动带入推荐 W/B 值。支持手动调整外加剂掺量。

**操作步骤**:

1. 点击推荐强度等级表格中的目标行（如 "≥120 MPa → 0.14-0.16"）
2. 系统自动带入推荐水胶比范围中间值

4. 查看系统自动计算的用水量
5. 点击 **"下一步"**

### 6.4 砂胶比页签

**页签名称**: 砂胶比

> [待截图: UHPC 砂胶比页]

**功能概述**:
输入砂胶比（S/B），系统结合前一页的水胶比参数计算钢纤维前的配比体积关系。

**操作步骤**:

1. 参考推荐值，输入砂胶比（S/B）
2. 系统自动计算砂的体积占比和关键体积参数
3. 点击 **"下一步"**

### 6.5 钢纤维用量页签

**页签名称**: 钢纤维用量

> [待截图: UHPC 钢纤维用量页]

**功能概述**:
输入钢纤维体积掺量，计算钢纤维质量以及考虑钢纤维影响的配比调整。

**操作步骤**:

1. 输入钢纤维体积掺量（通常 2%-4%）
2. 输入钢纤维密度（默认 7850 kg/m³）
3. 系统自动计算钢纤维质量

### 6.6 胶凝材料比例页签

**页签名称**: 胶凝材料比例

> [待截图: UHPC 胶凝材料比例页]

**功能概述**:
输入粒径参数（D10/D50/D90）、微粉参数和材料密度，系统通过修正安德森模型自动计算胶凝材料体积比例、质量比例及每方用量。同时根据强度等级显示微粉系数参考提示。

**输入参数**:

| 参数                   | 说明               |
| ---------------------- | ------------------ |
| 水泥体积分数           | 可手动或自动计算   |
| 硅灰体积分数           | 可手动或自动计算   |
| 微粉体积分数           | 可手动或自动计算   |
| 各材料密度             | 实测值             |
| 粒径 D10/D50/D90       | 各材料粒径分布参数 |
| 微粉参数（分布模量等） | 影响粒径累计分布   |

**操作步骤**:

1. 依次输入各胶凝材料的粒径参数（D10, D50, D90）
2. 输入各材料密度
3. 输入微粉参数
4. 系统自动计算：
   - 胶凝材料体积比例
   - 初始质量比例与修正质量比例
   - 每方材料用量与总量
5. 检查计算结果，确认无误

## 7. 试配调整 — 高性能试配（HPC）

### 7.1 进入 HPC 试配页

**入口**: 导航栏 → "试配调整" → "高性能试配"

**前置条件**:

- 试配页会自动读取当前已打开（或最近操作）的 HPC 主配比数据
- 页面顶部显示关联的项目名称和配比记录名称

- 右侧侧栏显示试配汇总

### 7.2 工作性实验页签

**页签名称**: 工作性实验

> [待截图: HPC 试配 — 工作性实验]

**功能概述**:
在基准配比基础上，调整胶材用量、砂率和外加剂掺量的增减量，系统自动重算试拌配合比和试拌用量。

**可调参数**:

| 参数                | 说明                 | 单位  |
| ------------------- | -------------------- | ----- |
| 胶材用量增减量 Δmb  | 相对基准配比的调整量 | kg/m³ |
| 砂率增减量 Δβs      | 相对基准配比的调整量 | %     |
| 外加剂掺量增减量 Δα | 相对基准配比的调整量 | %     |

**操作步骤**:

1. 在基准配比调整区输入各项增减量
2. 系统自动（经 500ms 防抖）重新计算并显示：
   - 试拌配合比（各材料每方用量）
   - 试拌用量（按试拌体积折算）
3. 底部参考区域显示 HPC 工作性评价指标
4. 调整满意后点击 **"下一步"**

### 7.3 强度实验页签

**页签名称**: 强度实验

> [待截图: HPC 试配 — 强度实验]

**功能概述**:
在工作性配比基础上生成不同水胶比的试验组，录入各组实测强度后进行二次回归分析，生成推荐水胶比。

**操作步骤**:

1. 查看系统自动生成的试验配比组（+Δ 组和 -Δ 组）
2. 对每组录入实测的混凝土强度值
3. 系统自动执行回归分析，显示：
   - 回归系数 a、b
   - 相关系数 R²
   - 推荐水胶比和预测强度
   - 回归曲线图表
4. 记录推荐值，点击 **"下一步"**

### 7.4 配合比校正页签

**页签名称**: 配合比校正

> [待截图: HPC 试配 — 配合比校正]

**功能概述**:
根据实测表观密度进行配合比校正，生成最终实验室配合比。

**操作步骤**:

1. 输入拌合物表观密度实测值
2. 系统自动计算校正系数 δ
3. 系统自动生成校正后的实验室配合比（各材料每方用量）
4. 确认无误后点击 **"保存配比记录"**

### 7.5 保存试配记录

> [待截图: HPC 试配保存对话框]

## 15. 推荐使用流程

对于日常业务，建议采用以下操作顺序：

1. **登录系统** → 使用分配的用户名和密码登录
2. **新建项目** → 在"项目管理"中创建项目，填写项目编号和名称
3. **进入配比计算** → 从项目详情页新建 HPC 或 UHPC 配比
4. **完成主配比** → 按页签顺序输入所有参数，保存配比记录
5. **试配调整（可选）** → 如需优化方案，进入"试配调整"页：
   - HPC：工作性 → 强度回归 → 配合比校正
   - UHPC：工作性 → 强度回归 → 校正与确定
6. **生成计算书** → 进入"计算书"页面，选择项目并生成可打印的计算书
7. **管理历史数据** → 在"全部配合记录"中统一查询和载入历史方案
8. **回收站管理** → 定期清理或恢复回收站中的数据
9. **维护个人资料** → 在"个人中心"修改密码和联系方式
10. **管理员维护用户** → 在"系统设置"中管理用户账号

## 16. 注意事项与常见问题

### 16.1 通用注意事项

1. 所有业务页面依赖登录态，未登录会自动跳转到登录页
2. 项目是数据组织的核心入口，建议始终先建项目再做配比
3. HPC 与 UHPC 配比数据自动按类别区分存储
4. 试配记录依附于各自的主配比记录保存
5. 删除操作（项目/记录）为软删除，数据进入回收站可恢复
6. 彻底删除（回收站中清空）为物理删除，不可恢复

### 16.2 HPC 计算注意事项

1. 建议按页签顺序逐页操作，前一页的结果会影响后续计算
2. 胶凝材料 28d 强度 fb 可用手动输入或自动估算两种方式
3. 回归系数 αa/αb 默认按规范取值，有试验数据时可上传拟合
4. 砂率需手动查表输入，建议参照交叉高亮推荐范围
5. 掺合料掺量之和不应超过合理范围

### 16.3 UHPC 计算注意事项

1. 强度等级行可点击快速带入推荐 W/B
2. 外加剂掺量如未输入，系统自动补入默认值
3. 胶凝材料比例页提供微粉系数参考提示
4. 粒径参数（D10/D50/D90）对安德森模型计算结果影响显著

### 16.4 试配注意事项

1. 必须先保存主配比记录才能进入对应试配页面
2. 试配结果不会自动覆盖主配比，保存时以试配快照形式附加
3. UHPC 试配校正提供三种基准选择，请根据实际情况选取
